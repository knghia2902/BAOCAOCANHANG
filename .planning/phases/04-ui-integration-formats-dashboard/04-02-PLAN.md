    If `activeTab` is 'pdf' or 'ocr':
      Call `await pdfOcrService.process(file, { targetFormat: targetFormat.value, useOcr: activeTab.value === 'ocr', onProgress: updateProgress })`.
      If successful, create a Blob from `result.data` with `result.mimeType`, trigger a download using `document.createElement('a')`, and show a success toast.
    If `activeTab` is 'convert':
      If file name ends with `.docx`, call `await documentConversionService.processDocxToPreview(file, previewContainer.value!, { onProgress: updateProgress })`.
      If file name ends with `.xlsx`, call `await documentConversionService.processXlsxToPreview(file, previewContainer.value!, { onProgress: updateProgress })`.
      If successful, call `await documentConversionService.exportToPdf(previewContainer.value!, file.name, { onProgress: updateProgress })` to trigger the PDF download.

## Threat Model
<threat_model>
- File extension spoofing: Services perform internal structural parsing which acts as a secondary verification, mitigating basic extension spoofing.
- XSS via DOM Preview: `DocumentConversionService` explicitly handles HTML sanitization during table building, and `docx-preview` is a trusted renderer. No raw text is dumped to the UI directly.
</threat_model>

## Verification
<must_haves>
- `PdfOcrTools` renders on the `/tools` page.
- Tabs correctly toggle `activeTab` states.
- End-to-end file selection routes to the correct service and downloads the output file.
- Error toasts correctly fire when a file > 15MB is uploaded.
</must_haves>
