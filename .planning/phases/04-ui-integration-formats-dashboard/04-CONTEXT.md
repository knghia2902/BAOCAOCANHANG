# Phase 4 Context

## User Decisions
- **Giao diện (UI):** Kế thừa phong cách hiện có của ứng dụng (Tailwind, thiết kế hiện đại, đồng nhất).
- **Tiến trình (Progress):** Thanh tiến trình (Progress bar) hiển thị % và trạng thái.
- **Xử lý lỗi:** Hiển thị thông báo (Toast notification) ở góc màn hình.
