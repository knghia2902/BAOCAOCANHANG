  <name>Task 2: Configure Vite build target and TS types</name>
  <files>vite.config.ts, tsconfig.app.json</files>
  <action>
    Cập nhật vite.config.ts để cấu hình build target và optimizeDeps target thành 'es2022' (hoặc 'esnext') nhằm hỗ trợ các trường class private và top-level await từ pdfjs-dist.
    Cụ thể, thêm:
    ```typescript
    build: {
      target: 'es2022'
    },
    optimizeDeps: {
      esbuildOptions: {
        target: 'es2022'
      }
    }
    ```
    Trong tsconfig.app.json, đảm bảo target hoặc lib tương thích với es2022 trở lên.
  </action>
  <verify>
    <automated>npm run build</automated>
  </verify>
