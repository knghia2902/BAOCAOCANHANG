# Plan 03-01: DOCX to PDF Pipeline Infrastructure

## Goal
Build the foundation for document conversion and the DOCX to PDF pipeline by defining types and implementing the `DocumentConversionService` with `docx-preview` and `html2pdf.js` integration.

## Requirements
- CONV-04: User can upload a Word (.docx) file, render a preview in the browser DOM, and convert it to a downloadable PDF file client-side.
