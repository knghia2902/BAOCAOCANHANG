    1. Tạo file `src/services/pdf/DocumentBuilder.ts`.
    2. Triển khai thuật toán gom cụm tọa độ X (1D Clustering) trong `DocumentBuilder.ts`:
       - Hàm gom cụm X: thu thập tất cả tọa độ X từ các phần tử chữ trong tài liệu, sắp xếp tăng dần.
       - Gom nhóm các tọa độ X có khoảng cách nhỏ hơn `clusterThreshold = 12` đơn vị.
       - Tính trung bình X của mỗi cụm để đại diện cho tọa độ xuất phát của cột, sau đó sắp xếp tăng dần.
       - Ánh xạ mỗi `TextElement` trên dòng vào cột gần nhất của nó. Ghép văn bản nếu có nhiều phần tử rơi vào cùng một cột.
    3. Triển khai phương thức `async buildExcelDocument(lines: LineGroup[]): Promise<ArrayBuffer>`:
       - Khởi tạo `ExcelJS.Workbook()`.
       - Thêm sheet "Extracted Text".
       - Điền mảng 2D sau khi gom cột vào bảng tính.
       - Duyệt qua các cột, tự động điều chỉnh độ rộng cột theo công thức: `column.width = Math.min(maxLength + 2, 50)` để tránh tràn ô dữ liệu.
       - Ghi ra buffer qua `await workbook.xlsx.writeBuffer()`.
    4. Viết file test `src/services/pdf/__tests__/excel-builder.spec.ts`:
       - Mock dữ liệu tọa độ X phân tán để kiểm tra thuật toán phân cụm.
       - Đảm bảo các ô văn bản được đặt đúng cột trong lưới 2D.
       - Chạy và xác minh buffer Excel được sinh ra thành công.
  </action>
