# Rules

## Git Workflow
- Sau khi hoàn thành chỉnh sửa mã nguồn hoặc hoàn thành yêu cầu của người dùng, hãy luôn tự động thực hiện commit và push các thay đổi lên GitHub:
  1. Chạy `git add .` để chuẩn bị các file thay đổi.
  2. Thực hiện `git commit -m "feat/fix/chore: <mô tả ngắn gọn về thay đổi>"` với thông điệp rõ ràng.
  3. Chạy `git push origin master` (hoặc nhánh đang phát triển) để đẩy mã nguồn lên GitHub.
